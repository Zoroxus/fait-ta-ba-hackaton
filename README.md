# Mimo
Mimo est un projet d'application mobile cherchant à aider des personnes agées isolées et des bénévoles (ou vice versa) à rentrer en contact pour des activitées.
Développé par Baptiste (Chef de Projet), Jonathan (Dev Front-End), et Maxime (Dev Back-End) avec react, node.js et mongodb.
